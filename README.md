# Samaritan sddm Theme

A minimal and simple sddm theme inspired by Samaritan from the CBS show "Person of Interest", the base of the theme was 
built on the [Void SDDM Theme](https://github.com/talyamm/VoidSDDM) built by [talyamm](https://github.com/talyamm). So big credit to him,
go like his project too if you like this.
